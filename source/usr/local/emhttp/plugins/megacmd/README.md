**MEGAcmd**

MEGAcmd brings [MEGA](https://mega.nz) cloud storage sync to Unraid, powered by MEGA's official command-line client.

Log in to your MEGA account and configure two-way syncs between MEGA folders and any Unraid share, all from Settings > Utilities > MEGAcmd. No terminal needed -- sync setup, current syncs, sync history, and MEGAcmd version updates are all managed from the webGUI.

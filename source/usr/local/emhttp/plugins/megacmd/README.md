**MEGAcmd**

MEGAcmd brings [MEGA](https://mega.nz) cloud storage sync to Unraid, powered by MEGA's official command-line client.

Log in to your MEGA account and configure two-way syncs (or one-way backups) between MEGA folders and any Unraid share, all from Settings > MEGAcmd. No terminal needed -- sync setup, sync history, notifications, a crash watchdog, speed limits, and MEGAcmd version updates are all managed from the webGUI.
